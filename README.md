# 📊 Customer Churn Analysis & Prediction

## 📌 Overview
This project analyzes telecom customer data to identify churn patterns and build a predictive model to detect customers at risk of leaving.

Customer churn is a critical business problem as it directly impacts revenue and increases customer acquisition costs.


## 🎯 Objectives
- Analyze key factors influencing customer churn  
- Identify high-risk customer segments  
- Build a predictive model to detect churn  
- Provide business-focused recommendations  


## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib & Seaborn  
- Scikit-learn  


## 📊 Key Analysis Performed
- Churn distribution analysis  
- Contract type vs churn  
- Tenure vs churn  
- Monthly charges vs churn  
- Customer risk segmentation using churn score  
- CLTV (Customer Lifetime Value) analysis  


## 🔍 Key Insights
- Customers with shorter tenure are more likely to churn  
- Month-to-month contracts show the highest churn rates  
- Higher monthly charges increase churn probability  
- High churn score customers are significantly more likely to leave  
- Loss of high-CLTV customers creates major business impact  


## 🤖 Model Performance
**Logistic Regression (Balanced)**  
- Accuracy: ~75%  
- Improved detection of churned customers (better recall)

**Random Forest**
- Accuracy: ~79%  
- Strong overall performance with better balance


## 💡 Business Recommendations
- Improve early-stage customer experience (first few months)  
- Encourage long-term contracts to reduce churn  
- Monitor high monthly charge customers for dissatisfaction  
- Target high-risk customers using churn score  
- Prioritize retention of high-value (CLTV) customers  

## 🚀 Outcome
This project demonstrates how data analytics and machine learning can be used to identify churn drivers and support data-driven retention strategies.
